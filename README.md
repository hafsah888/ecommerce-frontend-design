# Ecommerce Frontend Design

A responsive multi-page eCommerce website built using HTML, CSS, and JavaScript. The project focuses on creating a modern online shopping interface with product browsing, filtering, product details, and cart management features.

## Features

### Home Page

* Hero banner section
* Category navigation menu
* Deal countdown timer
* Product showcase sections
* Inquiry form
* Recommended products section
* Supplier by region section
* Newsletter subscription form

### Product Listing Pages

* Grid and List view layouts
* Product filtering system
* Price range filter
* Brand and category filters
* Rating filters
* Sort functionality
* Pagination

### Product Detail Page

* Product image gallery
* Product information and specifications
* Tiered pricing section
* Product tabs (Description, Reviews, Shipping, Seller Info)
* Related products section
* Add to Cart functionality

### Shopping Cart

* Update item quantities
* Remove items from cart
* Save items for later
* Coupon code support
* Order summary calculation
* Checkout section

### Responsive Design

* Mobile-friendly layout
* Hamburger navigation menu
* Tablet and desktop support
* Responsive breakpoints for different screen sizes
